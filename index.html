The git hub three basic actions that every developer needs to know.

  • Modified
  • Staged
  • Commit

The modified

This is when you have newly applied a change to your file ready to be stagged.

The staging area

This is the area where the file is ready to be committed for git and github to recognise the changes that has been made in order to run the commit message.

The commit

The commit is when you have committed the changes you have made into the github local or the remote repository.
